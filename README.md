# LocalGigs 🎶

LocalGigs is a mini catalog for local concerts and performances by independent musicians.  
The platform helps people discover small local gigs, while musicians and venue owners can get more visibility.

## Business model

Basic listings are free. Musicians, bands, or venue owners can submit their event for free.  
They can also pay for a Featured placement, which makes their event appear first on the homepage and visually highlighted.  
This works because local performers need attention, and users want an easy way to find interesting events nearby.

## Tech stack

- Python
- Flask
- PostgreSQL
- Docker Compose
- HTML/CSS
